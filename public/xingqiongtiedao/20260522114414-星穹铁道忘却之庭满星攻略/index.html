忘却之庭满星？老东西带你抄作业，平民阵容照样乱杀

崩坏星穹铁道忘却之庭满星攻略，三年老玩家手把手教你平民阵容搭配、实战技巧与资源规划，告别卡关，轻松拿满星琼。

玩了这游戏三年多，从开服被忘却之庭教做人，到现在每期稳定满星，中间也是踩了不少坑。贴吧群里天天有人问“我这阵容怎么打不满”、“XX角色到底值不值得练”，今天干脆把我的经验全倒出来，希望能帮到还在挣扎的兄弟们。咱们不谈那些虚的，就讲怎么用你手头的角色，把这星穹铁道忘却之庭满星攻略给它吃透。

抄作业时间：那些年我用过的神仙阵容#

别信什么“没有XX角色就别想满星”的鬼话，我开服那会儿用御三家加个白露不也磕磕绊绊打过去了？当然，现在角色多了，选择也多，配队核心思路就俩：“一队通解”或者“对策卡拆解”。

如果你是刚能打忘却之庭的萌新，我强烈推荐先练好一队能应对大部分情况的“万金油”。比如经典的“景元+停云+艾丝妲+奶妈”。这队强在哪？景元神君是独立行动，不吃速度，艾丝妲加攻加火伤，停云充电宝，对单对群都不虚。我小号就这么玩的，哪怕景元练度一般，靠神君刮痧也能把一些层数磨过去。

另一个思路就是针对性拆解了。忘却之庭上下半经常有不同弱点，你需要根据怪物弱点灵活变阵。这里给个我目前用得最顺手的“平民战神”列表：

  • 对量子弱点：希儿/青雀 + 银狼（没有银狼就用对应弱点的副C）+ 停云 + 奶
  • 对虚数弱点：饮月/瓦尔特 + 驭空/艾丝妲 + 同谐 + 奶
  • 火队（打冰胖、药王）：姬子/虎克 + 艾丝妲 + 火主/存护 + 奶
  • 雷队（打雷弱多环境）：卡芙卡/景元 + 希露瓦/黑天鹅 + 奶 + 盾/辅

记住啊，没有绝对的通解队伍，只有灵活的开拓者。多看看怪物弱点说明，比你无脑堆伤害管用。

别光看面板，这些细节才是满星关键#

角色拉到80级，行迹点满，遗器刷到“能用”就完事了？大错特错！忘却之庭打不满，很多时候就差在细节上。

速度和行动顺序是爹。你可能没注意，但角色出手顺序直接影响循环。比如，你想让希儿在第一回合就吃到艾丝妲的增伤，那艾丝妲的速度就必须比希儿快！我以前就犯过这错误，辅助速度慢半拍，C位第一轮就是“裸奔”状态。实战中发现，把同谐角色的速度鞋拉到140以上，体验会有质变。

充能绳和能量循环。很多大招流角色（说的就是你，停云、艾丝妲），没有充能绳就是残疾。停云大招给队友回50点能量，如果自己充能不够，两回合开不出来一次大，队友的循环直接断掉。我试过给停云换充能绳，大招频率从3回合一次变成稳定2回合一次，整个队伍的输出节奏顺滑了不止一星半点。充能阈值这玩意儿，得自己在“模拟宇宙”里多打打木桩感受一下。

针对性带光锥和遗器。这期忘却之庭有“承伤增加”的词条？那你的存护角色（杰帕德、火主）可以考虑带“记忆的质料”而不是“制胜的瞬间”，白值减伤更实在。遇到会给自己上增益状态的BOSS（比如可可利亚），带个有驱散能力的角色（罗刹、佩拉），比你硬灌伤害有效率得多。一个驱散可能省下你两回合的输出时间。

老东西的血泪教训：这些坑你别踩#

看着别人满星截图流口水，自己一上手就翻车？来来来，看看这些经典错误你中了几条。

头铁硬打，不懂变通。这是新手最容易犯的错。一看对面是雷弱，就把景元卡芙卡拉上来莽，结果被对面的控制或者高伤AOE打得找不着北。比如可可利亚召唤的冰柱，你不先清掉，主C活不过两回合。记住，忘却之庭的挑战目标是“在X回合内通关”，而不是“打败BOSS”。有时候先清小怪、破盾，反而比盯着BOSS打更快。

资源太分散，雨露均沾。我朋友就是，看这个角色好看练一下，那个角色强又拉一下，结果一队主力都没拉满，二队更是残缺不全。前期资源（经验书、信用点、行迹材料）巨缺，你必须有所取舍。我的建议是：先保证两队各有一个绝对主力输出拉满（等级、行迹、关键遗器），然后再去考虑辅助的练度。一个80级超模的C，带三个70级的辅助，远比四个都是75级的半吊子强。

遗器死磕，本末倒置。多少体力浪费在“出货率”极低的遗器本里？说的就是“量子套”和“快枪手”。我的经验是，主词条对了（比如暴击衣、速度鞋、属性球、充能/攻击绳），就可以停手了！副词条有个双暴或者大攻击就是赚。把拉一个角色从“能用”到“极品”的体力，够你把另一个辅助的关键行迹从6级点到8级了，后者提升更稳定。当然，你是氪佬或者肝帝就当我没说。

忽视“通用辅助”的练度。停云、艾丝妲、佩拉、鸭鸭（布洛妮娅），这几个四星战神和五星真神，练了绝对不亏。他们的增伤、拉条、充能、减防能力是跨属性通用的。尤其佩拉，一个群体减防大招，在高压环境下收益可能比你多上一个副C还高。我现在的二队核心就是佩拉+银狼，减防叠起来，主C伤害数字看着都吓人。

满星不是终点，是新的开始#

打满忘却之庭最大的奖励，其实不是那几百星琼（虽然确实香），而是一种“我的BOX练度终于得到验证”的成就感。它逼着你去研究机制、优化配队、抠细节，这个过程本身就很有趣。

别太焦虑，这期打不满，可能是角色池还没到那一步。慢慢积累，看看攻略，多尝试。很多时候你离满星就差一个正确的配速，或者一个关键的行迹等级。

好了，絮絮叨叨说了这么多，核心思想就一个：理解你的角色，看懂关卡机制，然后灵活地揍它。去试试吧，调整一下你的队伍，说不定下一期结算邮件里，就有那梦寐以求的800星琼了。冲就完了！
